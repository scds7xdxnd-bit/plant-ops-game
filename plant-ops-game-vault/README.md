# Plant Operations Game Vault

Open this folder as an Obsidian vault:

`plant-ops-game-vault`

Start at:

`00-index/Home.md`

The vault currently contains early planning notes for:

- game vision, modes, and core flow for a chemical process engineering game
- simplified BoD/DBM to process design decision gameplay
- Solvex-A specialty chemical plant scope
- chemical engineering concept to decision mapping
- Gantt-style level progression and difficulty modes
- plant simulation model, metrics, formulas, and scenario schema for later expansion
- technology stack, source of truth, repository structure, and infrastructure decisions
- production Gantt chart, roadmap, MVP backlog, and decision records
- domain research, starter scenario, and reusable templates

Use `00-index/Vault Map.md` to see how the notes connect.
